# けん玉部 出欠管理アプリ

## 📌 概要
新入生の参加率が低い課題を解決するために開発した出欠管理Webアプリです。
LINEでの出欠管理ではデータ分析ができなかったため、自ら設計・実装しました。

## 🎯 目的
- 出席率を曜日別に分析
- 欠席理由を可視化
- 課題を定量的に把握する

## 🛠 使用技術
- HTML
- CSS
- JavaScript
- Chart.js
- Git / GitHub

## ✨ 主な機能
- 出席・欠席登録
- 欠席理由アンケート機能
- 曜日別出席率のグラフ表示
- 欠席理由の円グラフ表示
- CSVダウンロード機能

## 📊 工夫した点
- データを構造化して保存
- 分析ロジックを自作
- 可視化まで実装
